Boxing Manager — Product Requirements Document
==============================================
Legend: [x] = complete, [ ] = not started / in progress


CORE CONCEPT
------------
The user plays as a gym owner: recruit and train boxers, schedule fights, earn prize money and ad revenue,
and upgrade the gym. The goal is to develop fighters to become world champions.


GYM SYSTEM
----------
[x] Gym levels 1–10 (level 10 costs $100,000,000)
[x] Higher gym levels attract better recruits and coaches
[ ] All money flows to the gym (individual boxers earn nothing directly)
[x] Gym upgrade UI (Finances tab)


BOXER SYSTEM
------------
[x] Weight classes: Flyweight (126 lbs), Lightweight (135 lbs), Welterweight (147 lbs),
    Middleweight (160 lbs), Heavyweight (200+ lbs)

[x] Reputation levels (lowest → highest):
    Unknown, Local Star, Rising Star, Respectable Opponent, Contender,
    Championship Caliber, Nationally Ranked, World Class Fighter,
    International Superstar, All-Time Great

[x] Fighting styles (rock-paper-scissors counters):
    Out-Boxer → countered by Swarmer
    Swarmer   → countered by Slugger
    Slugger   → countered by Counterpuncher
    Counterpuncher → countered by Out-Boxer

[x] Style focus stats:
    Out-Boxer:      Jab, Cross, Head Movement, Guard, Positioning, Speed
    Swarmer:        Lead Hook, Rear Hook, Body Movement, Positioning, Endurance, Toughness
    Slugger:        Rear Hook, Uppercut, Power, Endurance, Recovery, Toughness
    Counterpuncher: Timing, Adaptability, Discipline, Head Movement, Body Movement, Speed

[x] Boxer stats (1–20 scale; natural talent raises cap to 25):
    Offense:  Jab, Cross, Lead Hook, Rear Hook, Uppercut
    Defense:  Head Movement, Body Movement, Guard, Positioning
    Mental:   Timing, Adaptability, Discipline
    Physical: Speed, Power, Endurance, Recovery, Toughness

[x] Natural Talents (randomly generated or earned via training)
[x] Injuries (from training or matches) — data model complete
[x] Titles held with date ranges — data model complete
[x] Professional record format:
    "Win: Zach Pitt - KO (Rear Hook) - Rd. 1 (1:34) - North America Boxing Federation - April 14 2026"

[x] Boxer detail view (click name in Roster → detailed stats page)
[ ] Reputation increases as boxer wins fights
[ ] Injury recovery tracking (days remaining)


COACH SYSTEM
------------
[x] Coach skill levels: Local, Contender, Championship Caliber, All-Time Great
[x] Coach data model (style specialty, assigned boxer)

[ ] Coach recruitment (same flow as boxer recruiting)
[ ] Coaches train specific style stats during the season (exp per day)
[ ] Pre-fight training: switches from exp gain to temporary 300% stat boost
    (boost scales with number of training days before the match)
[ ] Higher gym levels attract better coaches


FIGHT SYSTEM
------------
[x] Federations: North America BF, South America BF, African BF, European BF,
    Asia BF, Oceania BF, International BF (most prestigious)
[x] Each federation has titles for every weight class
[x] Fight data model (method, round, time, title fight flag, etc.)

[x] Each federation hosts 4 events per year
[x] Fight scheduling UI — select opponent + event slot
[x] Contract negotiation:
    [x] Guaranteed payout ($1k–$10k in $1k steps, $10k–$100k in $10k steps,
        $100k–$1M in $100k steps)
    [x] PPV/Network split percentage (player's share)
    [x] Title fight option (winner takes title)
    [x] Opponent AI: can counter-offer (lower payout), negotiate PPV %, or reject
        (rejects if challenger rep too low or offer too poor)
[ ] PPV network signup (after scheduling, before fight):
    [ ] Networks have base viewership; expect 60% of viewers to watch
    [ ] Title fights increase viewer percentage
    [ ] Player earns their contracted PPV % of network revenue
[ ] Fight outcome formula: 20% style matchup + 70% stats + 10% random
[ ] Post-fight: contract payout deposited to gym balance


WORLD GENERATION (First Launch)
--------------------------------
[x] Detect empty DB on first launch and auto-generate world
[x] Seed all 7 federations with prestige values (International BF = 10)
[x] MVP — Welterweight only:
    [x] ~10 welterweight fighters per federation
    [x] Pyramid reputation distribution per federation:
          All-Time Great: 0–1 (IBF only)
          International Superstar: 0–1
          World Class Fighter: 1
          Nationally Ranked: 1
          Championship Caliber: 1 (champion)
          Contender: 2
          Respectable Opponent: 2
          Rising Star: 1
          Local Star: 1
          Unknown: 1
    [x] IBF skews higher (All-Time Great + International Superstar guaranteed)
    [x] 1 welterweight title per federation; champion = Championship Caliber or higher
    [x] IBF champion = Nationally Ranked or higher
    [x] Procedural names drawn from country pools per federation
    [x] Stats scale with reputation; style focus stats skew high
    [x] Natural talents: ~25% of fighters
    [x] Generated fight records consistent with reputation level

[ ] Future phase: generate all 5 weight classes per federation (~50 fighters total per federation)
